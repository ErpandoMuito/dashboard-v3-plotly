# Dashboard V3 - Tiny ERP Integration

Dashboard em Python usando Plotly Dash para integração com Tiny ERP.

## Instalação

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Execução Local

```bash
python app.py
```

Acesse: http://localhost:8050
